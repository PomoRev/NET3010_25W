TRouBLe

TRouBLe is not only a great way to remember the order of block element values (Top, Right, Bottom, Left), it also is a guide to the shorthand versions of these values.

One Value Applied to all four sides
Two Values First value is applied to Top and Bottom and second value is applied to Right and Left
Three Values Top takes the first value, Right and Left take the second value, and Bottom takes the last value.
Four Values Each side has its own value.

This applies to the shorthand attributes of border, margin, and padding.

Let's create an example of each using a 2, 6, 10, and 15 pixel values.

This order holds true in CSS until we get to grid and positioning elements on the grid.
